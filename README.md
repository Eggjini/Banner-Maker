## 기능 (Features)
- 주요 제목, 부 제목 입력 기능
- 글꼴 변경 기능
- 글자 크기 변경 기능
- 썸네일 배경색 변경 기능 (랜덤 추가)
- *썸네일 배경 그라데이션 추가 기능 (업데이트 중...)*
